## time.ly

timetable generator. to build and run, clone, `cd` into and run

```console
$ npm install
$ npm run dev
```

built in react with shadcn/ui.

### contributing

open an issue, then fork and submit a pull request. be respectful, write clean code, 
and wait for reviews.
